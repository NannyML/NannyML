.. _cli:

============================================
Command Line Interface (CLI)
============================================

``nml`` is a command line interface for using NannyML.
You can use it to run NannyML by providing some configuration without coding anything.

**usage**

.. code-block:: none

   nml [global flags] command [command flags]
