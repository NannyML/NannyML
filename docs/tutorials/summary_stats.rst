.. _summary-stats:

==================
Summary Statistics
==================

You can use NannyML to calculate summary statistcs of your data. There are five summary statistics available:

- Summation
- Average
- Standard Deviation
- Median
- Row Count

You can use the summary statistics to both perform simple drift checks as well as perform data quality checks
depending on your particular use case.
